# 📺 zenvideo: 1인 AI 방송국 운영 가이드

본 시스템은 주제 하나만으로 **대본 기획 → 성우 목소리 → 고품질 이미지 매칭 → 시네마틱 영상 조립**을 자동으로 수행하는 독립형 AI 영상 제작기입니다.

## 🚀 사용 방법 (How to Run)

터미널에서 아래 명령어를 입력하기만 하면 모든 공정이 시작됩니다.

```bash
npm run create
```

## 📂 결과물 확인 (Output)

영상 제작이 완료되면, 브라우저 링크를 찾을 필요 없이 **`output` 폴더**를 확인하십시오.

- 🎬 **output/victory.mp4**: 선명한 한글 자막과 효과가 적용된 최종 시네마틱 영상
- 🎙️ **output/voice.mp3**: 성우 나레이션 음성 파일

## ✨ 핵심 기술 특징 (Features)

1. **초정밀 한글 자막**: HTML 조립 엔진을 사용하여 어떤 환경에서도 한글이 깨지지 않고 선명하게 출력됩니다.
2. **시네마틱 효과**: 각 장면마다 부드러운 **줌인(Zoom In)**과 **페이드(Fade)** 효과가 자동으로 적용됩니다.
3. **물리적 배송**: 클라우드 링크의 불안정함(Access Denied 등)을 해결하기 위해, 모든 결과물은 사용자님의 하드디스크에 **물리적 파일**로 직접 저장됩니다.
